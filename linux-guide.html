Advanced Linux Guide

Slackware Linux

Slackware Linux, founded by Patrick Volkerding in 1993, is the oldest actively maintained Linux distribution. Known for its commitment to simplicity and stability, Slackware follows the KISS (Keep It Simple, Stupid) principle.

Core Principles

  • UNIX-like design philosophy
  • Stability and security focused
  • Manual dependency resolution for complete control
  • Plain text configuration files
  • BSD-style init scripts

Package Management

# Install a package installpkg package.tgz # Remove a package removepkg package # Upgrade a package upgradepkg package.tgz # Convert rpm to Slackware package rpm2tgz package.rpm

System Specifications

  • Latest Version: 15.0 (Released February 2, 2022)
  • Kernel Version: 5.15.x LTS
  • Package Format: .tgz, .txz (compressed tar archives)
  • Default Desktop: KDE Plasma 5.23.5
  • Alternative Desktops: Xfce 4.16, Fluxbox
  • Init System: SysVinit

Key Features

  • Full FHS (Filesystem Hierarchy Standard) compliance
  • GCC 11.2.0 compiler suite
  • Comprehensive development tools
  • PAM-free authentication
  • systemd-free operation

Linux Kernel

The Linux kernel, initially created by Linus Torvalds in 1991, is the core component of Linux operating systems. It serves as the main interface between hardware and system processes.

Kernel Architecture

  • Monolithic kernel with loadable module support
  • Preemptive multitasking
  • SMP (Symmetric Multiprocessing) support
  • Virtual memory management with paging
  • Unified VFS (Virtual File System) layer

Key Components

  • Process Scheduler: CFS (Completely Fair Scheduler)
  • Memory Management: NUMA, Transparent Huge Pages
  • File Systems: ext4, XFS, Btrfs, F2FS
  • Network Stack: TCP/IP v4/v6, Netfilter
  • Security: SELinux, AppArmor, Seccomp

Kernel Compilation

# Prerequisites apt install build-essential libncurses-dev bison flex libssl-dev libelf-dev # Download and extract kernel wget https://cdn.kernel.org/pub/linux/kernel/v5.x/linux-5.15.tar.xz tar xf linux-5.15.tar.xz cd linux-5.15 # Configure kernel make menuconfig # Build kernel (j8 for 8 CPU threads) make -j8 make modules -j8 sudo make modules_install sudo make install

Bootloaders

LILO (Linux Loader)

LILO is a legacy bootloader that directly accesses the BIOS. While less common today, it's still used in some specialized systems.

# Example lilo.conf boot = /dev/sda map = /boot/map install = /boot/boot.b prompt timeout = 50 default = linux image = /boot/vmlinuz-5.15.0 root = /dev/sda1 label = linux read-only

GRUB2 (Grand Unified Bootloader 2)

GRUB2 is the modern standard bootloader for most Linux distributions, offering advanced features and flexibility.

# Example grub.cfg entry menuentry "Linux" { insmod ext4 set root=(hd0,1) linux /boot/vmlinuz-5.15.0 root=/dev/sda1 ro quiet splash initrd /boot/initrd.img-5.15.0 }
